# C#遍历枚举

## 引言
这是一个常用到的问题，例如将枚举绑定到一个DropdownList……

我是这样实现的：
```       
  public enum Word
    {
        A,
        B,
        C,
        D,
        E
    }

  void Print()
        {
            foreach (Word w in Enum.GetValues(typeof(Word)))
            {
                Console.WriteLine((int)w+ ":" + w);
            }
        }
```
